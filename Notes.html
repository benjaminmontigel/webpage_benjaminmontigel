Part 1: Learn to Code

The World Wide Web

The web is composed largely of HTML documents. HTML stands for HyperText Markup Language.

The components of the web include the client, the internet, and servers.
The client and servers interact through HTTP. HTTP means Hypertext Transfer Protocol

HTML Basics

HTML elements are what we use to tell a web browser how to display content on a web page.This video gives a good introduction.

Some elements accept additional values called attributes in their opening tags.
Attributes provide additional information to the browser. For example, the anchor tag takes an attribute called href that defines an URL to link to.

HTML elements are either inline or block. Block elemnts form an invisible box around the content inside of them.Inline elements include the anchor tag, the span tag, the image tag, the strong tag and the line break tag.

Span vs. div: Span and div are both normal elements that can contain content. The only difference between these two is, span is inline, and div is block.

Part 2: Making a Stylish Webpage

CSS basics

While HTML controls the structure of a webpage, CSS controls the style. When programmers talk about the "DOM", which means Document Object Model, they are talking about the tree like structure of a page.

How to position boxes: Flexbox-Layout

Div's are box-elements, so by default they take up the entire width of the page.
Adding the rule display:flexwill override this behaviour.

Easy sizing of boxes

The size of an element is the sum of the border, padding and content width.
To make things easier use the box-sizing: border-box rule.
Just set the width of the overall box, and no matter how you tweak the padding and border sizes,
the size of the box will always stay the same.
Another useful thing to do is to set a max-width. The box becomes smaller when necessary,
but not larger than the set max width.
